An upstart's guide to blast

These are fast steps to run NCBI BLAST

Installation

To start you need to install NCBI's Blast program - Blast(p) for protein, Blast(n) for nucleotide

  1. You can access software here https://ftp.ncbi.nlm.nih.gov/blast/executables/blast+/LATEST/

  2. You need Blast(p) for protein

  3. Blast(n) for nucleotide

  4. Download ncbi-blast-2.13.0+-win64.exe (or whatever distribution is for your machine), run and install

    • or non-executable distribution and install through terminal

  5. Verify installation by using terminal, enter blastp, should say something in return regarding BLAST

  • NCBI BLAST PAGE

(LOCAL) Download a database

Download a database, in this example we will use swissprot: 

update_blastdb.pl --showall
In terminal will print a list of all available databases 
update_blastdb.pl --decompress swissprot
Will download swissprot database and decompress the file

(LOCAL) Navigate to data directory, use blastP to search for proteins

Download a database, in this example we will use swissprot: 

Blastp -db swissprot -query dog_tlrs.txt -out myfile2.tab

Here you are calling Blastp
Using the swissprot database with the -db flag
Providing a query list of sequences used to acquire hits from a database
and generating an output file called myfile2.tab
You need to navigate to output file (-out) and analyze results
Note that you can limit your search by using -taxidlist

  • More information

(REMOTE) Submit a remote query to NCBI

You can also search a remote database as follows with the -remote flag 

Blastp -db nr -query dog_tlrs.txt -out myfile2.tab -remote

Will perform your search. Here you are calling blastp
Using the swissprot database with the -db flag
Providing a query list of sequences used to acquire hits from a database
and generating an output file called myfile2.tab
You need to navigate to output file (-out) and analyze results
Note that you can limit your search by using -taxidlist
Additionally please note that some parameters aren't designed to work remotely

(WEB) Blastp in Web Browser

You can also perfom blast searches in your browser, though try to limit this to small searches
